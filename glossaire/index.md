# Glossaire

Voici une liste alphabétique des termes techniques utilisés dans cette documentation. Cliquez sur un terme pour en savoir plus.

:::accordion{title="Black Out"}
Mode d'urgence sur la console qui éteint instantanément tous les projecteurs. À activer avant d'éteindre la console et à désactiver au démarrage pour permettre l'utilisation normale.
:::

:::accordion{title="Chase"}
Fonction de poursuite automatique des scènes lumineuses. Peut être en mode Single Chase ou Mix Chase.
:::

:::accordion{title="Console lumière"}
Appareil central de contrôle des éclairages, permettant de régler l'intensité des projecteurs via des potentiomètres et boutons.
:::

:::accordion{title="Côté cour"}
Dans le jargon théâtral, désigne le côté droit de la scène vue depuis la salle (opposé au côté jardin).
:::

:::accordion{title="Côté jardin"}
Dans le jargon théâtral, désigne le côté gauche de la scène vue depuis la salle.
:::

:::accordion{title="Crossfade"}
Transition progressive entre deux presets en ajustant simultanément les masters A et B.
:::

:::accordion{title="DEL (Diode ÉlectroLuminescente)"}
Petite lumière LED indiquant l'état des modes ou boutons sur la console.
:::

:::accordion{title="Dispatcher"}
Boîtier de distribution qui relie les projecteurs au réseau DMX et gère leur alimentation électrique.
:::

:::accordion{title="DMX"}
Protocole de communication standard (DMX512) pour contrôler les équipements d'éclairage et de sonorisation dans les salles de spectacle.
:::

:::accordion{title="Estrade"}
Plateforme surélevée servant de scène dans la salle polyvalente.
:::

:::accordion{title="Fade"}
Potentiomètre contrôlant le temps de transition entre deux états lumineux, de 0 à 10 minutes.
:::

:::accordion{title="Flight Case"}
Boîte de transport robuste protégeant la console lors du stockage et du transport.
:::

:::accordion{title="Fondu"}
Voir Fade. Transition douce entre deux niveaux d'éclairage.
:::

:::accordion{title="Fondu au noir"}
Extinction progressive de toutes les lumières en descendant le master.
:::

:::accordion{title="Full On"}
Bouton pour allumer tous les projecteurs à pleine puissance tant qu'il est maintenu enfoncé.
:::

:::accordion{title="Interrupteur"}
Commutateur pour allumer ou éteindre la console.
:::

:::accordion{title="Lycée Val de Seine"}
Établissement scolaire où se trouve cette salle polyvalente équipée pour le théâtre.
:::

:::accordion{title="Master"}
Potentiomètre global contrôlant l'intensité de tous les projecteurs d'un preset. Il y a Master A et Master B, avec sens inversés.
:::

:::accordion{title="Mix Chase"}
Mode de poursuite où les scènes se mélangent automatiquement.
:::

:::accordion{title="Mode Select"}
Bouton pour choisir le mode de fonctionnement de la console, comme Double Preset.
:::

:::accordion{title="Pleins feux"}
Voir Full On. Allumage maximal de tous les projecteurs.
:::

:::accordion{title="Plateau"}
Surface de jeu sur l'estrade, équivalent à la scène.
:::

:::accordion{title="Potentiomètre"}
Manette rotative ou glissière pour ajuster l'intensité lumineuse.
:::

:::accordion{title="Preset"}
Configuration enregistrée des niveaux d'éclairage pour un groupe de projecteurs. Il y a Preset A et Preset B.
:::

:::accordion{title="Projecteur"}
Appareil d'éclairage directionnel utilisé pour créer l'ambiance lumineuse sur scène.
:::

:::accordion{title="Protocole DMX512"}
Standard technique pour la transmission de signaux de contrôle aux équipements scéniques.
:::

:::accordion{title="Régie audiovisuelle"}
Partie de la documentation consacrée au son et à la vidéo.
:::

:::accordion{title="Régie lumière"}
Contrôle et gestion de l'éclairage scénique.
:::

:::accordion{title="Salle polyvalente"}
Espace multifonction du lycée Val de Seine utilisé pour les spectacles et événements.
:::

:::accordion{title="Single Chase"}
Mode de poursuite où une seule scène est jouée à la fois.
:::

:::accordion{title="Technique"}
Ensemble des équipements et méthodes utilisés pour la production théâtrale (lumière, son, vidéo).
:::

:::accordion{title="Transition"}
Changement d'un état lumineux à un autre, manuel ou automatique via le fade.
:::

:::accordion{title="XLR"}
Type de connecteur utilisé pour les câbles DMX et audio.
:::
